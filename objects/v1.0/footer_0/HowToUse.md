# EYE.JS

### How to use

<br>
The <code> data </code> attribute expects a json object.<br>
<br>

```HTML
<eye type="footer_0" src="../footer_0/object.html" data='{
        "background": "black",
        "footer": [
            {
                "title": "Information",
                "links": {
                    "About Us": "#",
                    "Store": "#",
                    "Privacy Policy": "#",
                    "Terms & Conditions": "#"
                }
            },
            {
                "title": "Customer Service",
                "links": {
                    "Contact Us": "#",
                    "Site Map": "#"
                }
            }
        ],
        "note": "Made in Costa Rica<br>EyeJS 2021"
    }'></eye>
```
