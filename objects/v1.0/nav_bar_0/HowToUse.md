# EYE.JS

### How to use

<br>
The <code> data </code> attribute expects a json object.<br>
<br>

```HTML
<eye type="eye_nav_bar_0" src="./object.html" data='{
        "links":{
            "Key 1": "?Custom_url_1",
            "Key 2": "http://example.com/?value=2",
            "Key 3": "http://example.com/?value=3",
            "Key 4": "#Custom_url_4"
        },
        "background": "#000"
    }'></eye>
```
